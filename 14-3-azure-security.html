Question: "How does Azure's security model differ from AWS?"

Answer: "Azure's security is tightly integrated with the Microsoft ecosystem, especially around identity. Entra ID (Azure AD) provides enterprise-grade identity with features like Conditional Access and Privileged Identity Management built-in. Microsoft Defender for Cloud combines CSPM and workload protection in one service (vs. AWS's separate Security Hub and GuardDuty). Azure also has Microsoft Sentinel as a native cloud SIEM, while AWS relies on third-party solutions. The network model is similar - NSGs are like Security Groups, Azure Firewall is like Network Firewall. A key advantage of Azure is the integration with Microsoft 365 and Defender XDR for end-to-end visibility."
