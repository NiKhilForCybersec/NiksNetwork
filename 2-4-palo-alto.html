âœ… Best Practice - Application-Default Service

Always use "application-default" for the service when possible.

This setting allows the application only on its standard ports. For example:

  • DNS (app) with application-default = Only allows DNS on UDP/53, TCP/53
  • web-browsing with application-default = Only allows on TCP/80
  • ssl with application-default = Only allows on TCP/443

This prevents application evasion where malware uses non-standard ports.

Common Question: "What is HIP (Host Information Profile) in GlobalProtect?"

Answer: HIP collects information about the connecting device's security posture, such as:

  • Is antivirus installed and up-to-date?
  • Is disk encryption enabled?
  • Is host firewall running?
  • What OS version/patch level?
  • Is the device domain-joined?

This information can be used in security policies to allow/deny access or apply different policies based on device compliance. For example, non-compliant devices might only get access to a remediation portal.
