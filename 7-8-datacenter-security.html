Question: "How would you secure a data center network?"

Answer: "I'd implement defense in depth with focus on both north-south and east-west traffic. For north-south: edge firewalls with DDoS protection, DMZ for public services, and tiered zones (web, app, DB) with firewalls between them. For east-west: microsegmentation using NSX or similar to enforce workload-level policies - each server can only talk to specific servers on specific ports. This limits lateral movement if one server is compromised. I'd deploy IDS/IPS and NDR to monitor internal traffic, since that's where attackers move after initial breach. Management traffic would be on a separate isolated network with jump servers and PAM. All inter-tier communication would be encrypted with TLS. The key principle is zero trust internally - don't assume traffic inside the DC is safe."
